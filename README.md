# 🚀 LifeUp - 游戏化生活管理系统

> **期末考查项目**：将枯燥的待办事项转化为史诗般的 RPG 冒险！完成任务、赚取金币、升级技能，开启你的第二人生。

---

## 📖 项目简介

**LifeUp** 是一款基于 **Vue 3** 开发的游戏化任务管理应用。它打破了传统 Todo 软件的枯燥感，通过引入经验值（EXP）、金币（Gold）、等级（Level）和技能树系统，将个人的自我驱动转化为直观的游戏成长体验。

---

## 🛠️ 项目运行步骤 (重要：请老师查阅)

请确保您的计算机已安装 [Node.js](https://nodejs.org/) (建议版本 v18.x 或以上)。

### 1. 克隆或解压项目
如果您是从 GitHub 获取，请运行：
```bash
git clone https://github.com/你的用户名/LifeUp.git
cd lifeup
(如果是压缩包，请直接在解压后的 lifeup 文件夹中打开终端)
2. 安装项目依赖
在项目根目录下运行以下命令安装必要的库（Vue, Pinia, Lucide 等）：
code
Bash
npm install
3. 启动开发服务器
运行以下命令启动项目：
code
Bash
npm run dev
4. 访问应用
启动成功后，在浏览器中打开：http://localhost:5173 即可查看效果。
✨ 核心模块说明
📋 任务看板：支持不同难度的任务发布，通过拖拽管理任务状态。完成任务可获得金币与经验。
⏳ 专注地下城：番茄钟系统。选择不同难度的关卡进行专注，击败怪物并结算丰厚奖励。
🌳 无限技能树：8 种被动技能（如炼金术、寻宝、深度专注等），通过消费金币升级，全方位提升任务收益。
🛒 便利商店：用户可自定义现实奖励（如：玩一小时游戏），通过虚拟金币购买，实现良性激励。
📊 个人中心：记录等级成长、冒险日志、能力雷达图以及签到考勤日历。
🎨 设计亮点
响应式适配：专门针对移动端（底部导航栏）与桌面端（侧边栏）设计了两套交互布局。
主题切换：支持“奶油明亮模式”与“深夜咖啡模式”一键切换，全局变量控制色彩方案。
视觉素材：采用“线条小狗”系列可爱风素材，提升用户交互的亲和力与趣味性。
📚 技术栈
框架: Vue 3 (Composition API)
构建工具: Vite
状态管理: Pinia (实现用户存档与数据逻辑计算)
路由: Vue Router
图标: Lucide Vue Next
